# Vietmap React + Tailwind

Trang demo bản đồ Vietmap dựng bằng **React + Vite + Tailwind CSS**.

## 1) Cài đặt

```bash
npm install
```

## 2) Cấu hình API key

```bash
cp .env.example .env
```

Sửa `.env`:

```env
VITE_VIETMAP_API_KEY=your_real_api_key
```

## 3) Chạy local

```bash
npm run dev
```

Mở địa chỉ Vite hiển thị trong terminal (thường là `http://localhost:5173`).

## Build production

```bash
npm run build
npm run preview
```
